# SkillCraft-Technology_03_cat-dog-classification-svm
This project implements a Cat and Dog Image Classification system using a Support Vector Machine (SVM). Images are preprocessed with OpenCV, resized, converted into feature vectors, and classified using Scikit-learn. Model performance is evaluated using accuracy and a classification report.
# Cat vs Dog Image Classification using SVM

## Objective
Classify images of cats and dogs using a Support Vector Machine (SVM).

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- tqdm

## Dataset
Cats and Dogs image dataset.

## Steps
1. Load images.
2. Resize images to 64×64.
3. Flatten images into feature vectors.
4. Split dataset into training and testing sets.
5. Train an SVM classifier.
6. Predict image classes.
7. Evaluate using Accuracy and Classification Report.
